08 de Abril de 2026
Proyecto: Sistema Web de Reservas de Boletos
Hosting: DigitalOcean
Tecnologías: Django REST + React (Vite) + PostgreSQL
Región: New York (NYC1)
📋 Estado Actual del Sistema

El sistema de reservas se encuentra casi listo para despliegue. Módulos completados y funcionales:

✓ Registro e inicio de sesión (email + Google OAuth)
✓ Verificación de email con código de 6 dígitos
✓ Recuperación de contraseña
✓ Búsqueda y filtrado de viajes
✓ Selección interactiva de asientos con mapa visual
✓ Reserva múltiple con expiración a 15 min
✓ Subida de comprobante de pago
✓ Panel de admin para aprobar/rechazar pagos
✓ Generación de tickets con código QR
✓ Envío de boletos por email en PDF
✓ Verificación de tickets por QR
✓ Creación de viajes en lote
✓ API externa para sistema de control
✓ Tasa de cambio BCV automática
✓ Protección anti-spam y rate limiting
✓ Transacciones atómicas anti doble reserva
⏳ Módulo pendiente: Automatización de pagos con R4
Actualmente los pagos son validados manualmente. La integración con R4 permitirá verificar y confirmar pagos de forma automática, eliminando la intervención manual. Una vez completada, el sistema estará 100% operativo para producción.

✅ Recomendación Final

Para Aerorutas de Venezuela recomendamos el:

Presupuesto Recomendado — 34,20 US$/mes

Dominio recomendado: aerorutasdevenezuela.net (11,86 US$/año)

  1. La base de datos gestionada es crítica para un sistema de reservas donde la pérdida de datos es inaceptable.
  2. 2 GB RAM con Intel NVMe corre Django + Nginx holgadamente.
  3. 70 GB de disco NVMe da espacio suficiente para archivos y comprobantes de pago.
  4. El monitoreo gratuito permite detectar problemas antes de que afecten al servicio.
  5. Es fácil escalar al Presupuesto Profesional sin migración compleja.
  6. Costo razonable: aproximadamente 1,14 US$ diario.

📌 Notas Importantes
  1. Todos los precios son en dólares americanos (US$) y fueron verificados directamente en DigitalOcean (abril 2026).
  2. Región recomendada: NYC1 (New York) — la más cercana a Venezuela entre las opciones disponibles.
  3. Gmail SMTP tiene un límite de ~500 emails diarios (suficiente para las primeras etapas, pero considerar Mailgun si crece).
  4. Let's Encrypt se renueva automáticamente cada 90 días (gratis).
  5. Los precios de dominios son del proveedor actual del cliente.
  6. IVA/impuestos no incluidos.
  7. Todos los droplets usan CPU compartida (Shared CPU — Basic). Para CPU dedicada, los planes inician desde 63 US$/mes.
